# BRIDGE OS — Consola de Guardia

Aplicación web (PWA) de apoyo para el oficial de navegación / piloto costero.
Instalable en iPhone y Android, funciona sin conexión.

- **Checklist por fases** (pre-zarpe → emergencias)
- **Referencia rápida**: RIPA, IALA-B, señales sonoras, VHF (MAYDAY/PAN-PAN/SECURITÉ), Frases OMI (SMCP), nudos, banderas, unidades
- **Calculadoras náuticas**: calado/asiento, guinda, rumbos, velocidad·distancia·tiempo, mareas (regla de los doce), cadena de fondeo, escala de carta
- Tema **Día / Noche** · todo el progreso se guarda en el dispositivo

## Instalar en el teléfono
Abre la página en el navegador del celular y:
- **iPhone (Safari):** Compartir → *Añadir a pantalla de inicio*.
- **Android (Chrome):** menú ⋮ → *Instalar app*.

Hecho para la formación TUTMC (Transporte Marítimo y Costero).
